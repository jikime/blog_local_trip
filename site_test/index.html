    네이버 지도로 보기 (선택)

    네이버 지도 Maps API의 ncpKeyId를 입력하면 네이버 지도로 전환할 수 있어요. 같은 장소가 같은 방식으로 핀에 표시돼요.

    현재 환경에서는 네이버 API 키 입력이 불가능(준비 안 됨)하여 입력란을 비활성화했어요. 기본 동작은 키가 필요 없는 무료 지도(Leaflet + OpenStreetMap + Nominatim)로 제공돼요.

    동네 이야기

    동네마다 비슷한 수(세 곳)로 균형을 맞췄어요. 글 속 핀 번호를 누르면 위 지도에서 해당 장소로 바로 이동해요.

    소개

    동네 탐방은 한 사람이 직접 걸은 서울 동네의 기록이에요. 음식점·카페·공간·골목을 글과 위치로 남기고, 방문 전 동네 단위로 빠르게 둘러볼 수 있게 했어요. 별도 페이지 없이 이 한 페이지 안에서 모든 내용을 제공해요.

    주소와 영업 상태는 바뀔 수 있어요. 각 카드에 최종 확인일을 두었으니 방문 전 다시 확인하는 걸 권해요(콘텐츠 최신성 책임은 작성자에게 있어요).
